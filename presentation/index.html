Event Listener Syntax:

element.addEventListener(event, function, useCapture);

Event: this is the triggering condition. Examples include click, keyPress, or scroll

Function: this is a function defined elsewhere in your javascript, which gets called when the event happens

useCapture: this is a boolean value indicating whether the capture method is used to resolve event orders, or the bubble method (more on this later)

You can define the function in the event listener like this

element.addEventListener("click", function(){ alert("Hello World!"); });

or you can refer to a named function like this

element.addEventListener("click", myFunction); function myFunction() { alert ("Hello World!"); }
